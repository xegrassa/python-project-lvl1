## Description

The set of mini-games that are launched from the console.

### Install

`pip install -i https://test.pypi.org/simple/ --extra-index-url https://pypi.org/simple/ xegrassa-brain-games`

### Play

Type in terminal one of these games

- brain-even
- brain-calc
- brain-gcd
- brain-balance
- brain-progression
- brain-prime
